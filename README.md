# Network Monitoring Dashboard

Realtime network monitoring dashboard built with:

- FastAPI
- React (Vite)
- WebSocket
- SQLite

## Features

- Realtime device monitoring
- Online / Offline status
- Network scanner
- Remote VNC integration
- Dark mode dashboard

## Run Development

Backend

uvicorn app.main:app --reload

Frontend

npm run dev

## Production

Build frontend

npm run build

Run backend

uvicorn app.main:app
